# Cooking Instructions Skill

Use this skill when a recipe needs clear execution steps that are easy to follow in a kitchen.

## Workflow

1. Start with preparation actions before heating the pan or appliance.
2. Use short step titles and direct descriptions.
3. Assign an estimated time to each step.
4. Keep steps in the actual order a user should perform them.
5. Mention key timing, texture, or doneness cues when relevant.
6. Avoid vague instructions such as "cook until done" without a cue.
7. Keep safety and practicality in mind, especially for hot oil, knives, and raw meat.

## Output Shape

Each step should include:

- order
- title
- description
- timeMinutes
