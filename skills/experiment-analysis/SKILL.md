# Experiment Analysis Skill

Use this skill when a paper reading task requires deeper analysis of experimental design and empirical evidence.

## Workflow

1. Extract datasets, tasks, splits, evaluation metrics, and experimental settings.
2. Identify baselines and explain why they are relevant comparisons.
3. Summarize main quantitative and qualitative results.
4. Separate main experiments from ablations, sensitivity studies, and case studies.
5. Check whether the evidence supports the paper's claimed contributions.
6. Identify missing comparisons, weak baselines, possible data leakage, or insufficient evaluation.
7. Summarize threats to validity and external generalization.

## Output Shape

Prefer concise Markdown sections:

- Experimental Setup
- Datasets and Metrics
- Baselines
- Main Results
- Ablation / Sensitivity Analysis
- Evidence Strength
- Validity Concerns
