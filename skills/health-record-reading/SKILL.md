# Health Record Reading Skill

Use this skill when processing health checkup reports, lab result text, or health record files.

Always extract:

1. Source file and report title
2. Common health indicators and values
3. General interpretation notes
4. Questions to discuss with a licensed clinician
5. Non-diagnostic lifestyle information
6. A clear safety notice

Do not infer a diagnosis, prescribe medication, or suggest stopping medication.
